Software de versionamento:
Faz o controle de versões de um código ou documento, tornando possível gerenciar
as versões realizadas por um desenvolvedor ou uma equipe de desenvolvedores.
um software de controle de versão pode ser:

Centralizado
Possui um repositório central em um servidor
com controle de versões, dependente
do servidor.
Software Subversion
Distribuído
Possui um repositório local com controle de
versões.
Atualizações no repositório central independente
do servidor.
Softwares Mercurial, Git

Comandos GIT

commit
torna permanente um conjunto de alterações em
um repositório
push
Envia alterções para o servidor no repositório
remoto
fetch
Verifica se o seu repositório local está em
sincrionia com o remoto
pull
Sincroniza o repositório local com o remoto
atualizando as versões
